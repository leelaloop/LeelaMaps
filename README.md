This is an html UI for a new LeelaMaps
generated with Poe.

Features:
* map you can search
* user creation
* hold-tap to start a new note
* note templates (created by users)
* custom pins
* media gallery w/ optional titles, descriptions, quantities, and prices
* event time
* basic privacy settings

It needs:
* backend (react.js, python) + db (postgres, Neo4j)
* admin dashboard
* messager
* shopping cart for notes that have priced media
* legal language defining LeelaMaps as a map notes platform, and it is not responsible for sales on the platform; templates for legal guarantees
* saveable search layer filters + notification options
* adventure-mode: parabolic orientation following angles of mobile phones (tilt and direction); pull from other project, device-orientation.html
* approx. author distance from pin, note & comments
* user-set collections & hot buttons
* security features
* a team to keep developing it
